Project Overview
----------------
Develop a system to classify objects in RGB images.

Requirements
------------
- Python Version: 3.6 or higher
- Environment: Jupyter Notebook

Data Preparation
----------------
- The dataset is included in the provided zip file. Ensure that the dataset files are located in the same directory as your Jupyter Notebook.

Running the Project
-------------------
1. Launch Jupyter Notebook:
   - Open the file `main.ipynb` in Jupyter Notebook.
2. Execute Cells:
   - Run each cell in order to execute the code and view the results.
3. Follow Instructions:
   - Adhere to any additional instructions within the notebook for further configuration or steps.

Viewing Results
---------------
- Classification Outcomes:
  - Results will be displayed directly within the Jupyter Notebook.
- Model Evaluation:
  - Metrics and performance evaluations will be provided after running the relevant cells.
